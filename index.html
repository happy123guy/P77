Project on Console


Q1-

What is visual studio and how to set it up?

The Visual Studio IDE is a creative launching pad that you can use to edit, debug, and build code, and then publish an app

STEPS-
Open Visual Studio IDE software.
Create a new folder on your computer.
Open Visual Studio IDE and click on File > Open Folder, then browse and open the folder created in step 2.
Install Live-Server Extension.

Q2-

What is console screen and how to open it up?

A console screen is where a user can input commands and receive visual output from a computer or computer system

STEPS-
Open screen you are testing.
Right click and click inspect.
A pop-up will appear on the right, click console on the top.
